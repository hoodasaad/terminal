Terminal

Is a JQuery plugin running popup terminal to view any data like welcome messages, notifications , visits , etc.....

Docs Download .zip
Simple usage
    terminal({
      data: ["terminal-line 1","terminal-line 2","terminal-line 3","terminal-line 4","terminal-line 5"]
    });
Advanced usage
  terminal({
        id: 2017,
        data: ["terminal-line 1","terminal-line 2","terminal-line ...","terminal-line 50"],
        speed: 0.1,
        autoClose: true,
        title: "Terminal Title",
        title_bg_color: 'white',
        title_color: 'black',
        textColor: "#ffffff",
        beep: true,
        wait: 0,
        dir: 'ltr',
        position: 'topRight',
        background: '#432454',
        width: 450,
        height: 350,
        opacity: 0.9,
        border: 13,
        margin: 20,
        borderColor: '#432454',
        radius: 6
    },function(){
      console.log('Terminal Completed Callback_function()');
    }
  );
Terminal function properties
Property Value Description Default Importance
data array is simple array like examples that is shown in terminal ["Please set terminal data.."] required
id integer or text defines open terminal you can use to close it random number optional
speed integer milliseconds speed of writing lines in terminal 0.8 optional
wait integer seconds time in seconds before run terminal 0 optional
autoClose boolean automatically terminal close after finish all lines false optional
title text terminal title shown in head of terminal null optional
title_color Hex Color terminal title color if is set #1a1a1a optional
title_bg_color Hex Color terminal title background color if is set #bcbbbc optional
textColor Hex Color terminal line color #ffffff optional
beep boolean sound beep with every line written in terminal false optional
dir direction of terminal line text ltr optional
margin integer terminal margin from window screen 10 optional
position position of terminal topLeft optional
width integer terminal screen width 300 optional
height integer terminal screen height 200 optional
background Hex Color terminal screen background #000000 optional
border integer terminal screen border 3 optional
radius integer terminal screen border radius if set border 6 optional
borderColor Hex Color terminal screen border color if set border #c0b2b2 optional
opacity double from 0 to 1 terminal screen opacity 1 optional
CallBack function
   
    terminal({termianlOptions},YourCallback_function);
    Callback_function: will run after terminal finish all lines
    it should be written without brackets
Close Terminal
   
    close_terminal(TermianlID , AnimateClosing);
    TermianlID: is id property that you are entered in termianl function
    AnimateClosing: boolean (if true will animate terminal when close) -- default = false
